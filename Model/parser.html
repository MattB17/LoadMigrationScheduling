The Parser class is used to parse a file containing data specifying a migration instance.

Used to parse files containing a migration instance.

A dictionary of migrations. The keys are strings specifying the name of the switch involved in the migration and the keys are Migration objects representing the migrations.

A set of ControllerConstraint objects.
