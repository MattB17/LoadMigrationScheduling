Exceptions

A module containing the Exception classes generated when building a load migration schedule.

exception MigrationScheduling.exceptions.InstanceNotSpecified[source]

Exception raised when a load migration instance has not been specified.

exception MigrationScheduling.exceptions.InvalidName(message)[source]

Exception raised when an invalid object name is generated.

Parameters
message: str

A string representing the message displayed to the user.

exception MigrationScheduling.exceptions.ModelNotOptimized[source]

Generated when accessing details of model that has not been optimized.

exception MigrationScheduling.exceptions.UninitializedModel[source]

Exception raised when an model has not been initialized.

The Exception is raised when a user tries to access a model that has not yet been initialized.
